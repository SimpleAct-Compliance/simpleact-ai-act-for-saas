# SaaS Feature Inventory Template

Use this template to register customer-facing and internal AI features in a SaaS environment.

## Feature Identity

- feature name
- product area
- lifecycle status

## SaaS Context

- tenant scope
- customer-facing or internal
- deployment surface

## Ownership

- product owner
- technical owner
- compliance owner

## Dependencies

- provider
- model or API dependency
- infrastructure dependency

## Review

- last review date
- next review date
- release or reassessment trigger
